1. AUROC (Area Under ROC Curve)
================================================================================

AUROC measures how well the prefilter ranks anomalies above normal lines.
Score of 1.0 = perfect separation, 0.5 = random guessing.

┌──────────────────────────┬──────────┬──────────┐
│ Dataset                  │ Anomaly% │   AUROC  │
├──────────────────────────┼──────────┼──────────┤
│ 2000 normal + 50 anomaly │   2.5%   │  0.9998  │
│ 2000 normal + 100 anomaly│   5.0%   │  0.9982  │
│ 2000 normal + 200 anomaly│  10.0%   │  0.9991  │
│ 5000 normal + 100 anomaly│   2.0%   │  0.9999  │
├──────────────────────────┼──────────┼──────────┤
│ Average                  │          │  0.9993  │
└──────────────────────────┴──────────┴──────────┘

Interpretation: AUROC > 0.998 across all tests means the prefilter's
anomaly score almost perfectly separates real anomalies from normal log
lines. This is because anomalous lines (errors, security alerts, etc.)
have fundamentally different character-level structure from normal
access logs.

3. F1 SCORE AT DEFAULT SETTINGS (85th Percentile)
================================================================================

At the default 85th percentile threshold, 15% of lines are flagged.
The prefilter is intentionally generous (high recall, moderate precision)
since the LLM stage will filter false positives.

┌──────────────────────────┬──────────┬──────────┬──────────┬──────────┐
│ Dataset                  │ Anomaly% │    F1    │ Precision│  Recall  │

│ 2000 normal + 50 anomaly │   2.5%   │  0.2793  │  0.1623  │  1.0000  │
│ 2000 normal + 100 anomaly│   5.0%   │  0.4819  │  0.3175  │  1.0000  │
│ 2000 normal + 200 anomaly│  10.0%   │  0.7547  │  0.6061  │  1.0000  │
│ 5000 normal + 100 anomaly│   2.0%   │  0.2312  │  0.1307  │  1.0000  │
├──────────────────────────┼──────────┼──────────┼──────────┼──────────┤
│ Key Result               │          │          │ Recall = │   1.0    │
└──────────────────────────┴──────────┴──────────┴──────────┴──────────┘

IMPORTANT: Recall = 1.0 across ALL tests means the prefilter catches
100% of anomalies. Zero false negatives. This is the ideal behavior
for a prefilter — we never miss a real anomaly.

The precision is lower because we flag ~15% of lines but anomalies
are only 2-10% of the data. This is by design: the LLM will analyze
the flagged lines and filter out the false positives.
